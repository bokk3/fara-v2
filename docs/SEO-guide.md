## üîß Technische SEO Setup

### Sitemap Indienen

**Google Search Console:**
1. Ga naar https://search.google.com/search-console
2. Voeg je website toe (movetofit.be)
3. Verifieer eigenaarschap via DNS of HTML-bestand
4. Ga naar "Sitemaps" in het linkermenu
5. Voeg toe: `https://movetofit.be/sitemap.xml`
6. Klik op "Indienen"

**Bing Webmaster Tools:**
1. Ga naar https://www.bing.com/webmasters
2. Voeg je website toe
3. Verifieer eigenaarschap
4. Voeg sitemap toe: `https://movetofit.be/sitemap.xml`

### Robots.txt Verificatie

Controleer of robots.txt correct werkt:
- Bezoek: `https://movetofit.be/robots.txt`
- Test in Google Search Console onder "robots.txt Tester"

### Sitemap Bijwerken

Update `sitemap.xml` wanneer je:
- Nieuwe pagina's toevoegt
- Belangrijke content wijzigt
- De site structuur aanpast

Wijzig de `<lastmod>` datum naar de huidige datum.
